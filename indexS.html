Mis proyectos

Detector de Similitud de Código Fuente

¿De qué se trata?

Mide el nivel de similitud entre dos pares de código fuente. El programa toma como entrada dos o más archivos de código fuente y mide el nivel de similitud que existe entre cada par. La similitud varía del 0% al 100% si es exactamente igual.

¿Cómo se hizo?

El proceso consta de varias etapas:

  • Proceso de NLP: Tokenización, eliminación de comentarios, identificación de variables, eliminación de variables.
  • Construcción de una Bolsa de Tokens.
  • Modelo de Frecuencia de Término.
  • Coseno Vectorial.
  • Creación de la Matriz de Similitud.

Tecnologías utilizadas

Flask para el backend, HTML y CSS para la estructura y diseño web, y Python y JavaScript para la lógica y funcionalidad del programa.

Sistema de Gestión de Medicación para Enfermeros

¿Qué problema resuelve?

El sistema tiene una lista de medicamentos con su nombre, descripción y stock.

Existen cuatro roles: administrador, enfermero, farmacéutico y paciente.

¿Qué hace cada rol?

  • Administrador: Puede crear usuarios (enfermero, farmacéutico o administrador); puede gestionar a los usuarios (eliminar/editar); editar su perfil.
  • Enfermero: Puede realizar un pedido al farmacéutico, ingresando el ID del paciente y la lista de medicamentos solicitados.
  • Farmacéutico: El farmacéutico tiene la lista de peticiones que necesita ejecutar. Una vez que la lista esté lista, cambia de estado a 'entregar'. Una vez se entregue la petición, cambia de estado a 'entregado'.
  • Paciente: Puede ver las recetas médicas.

Tecnologías utilizadas:

  • Flask
  • HTML
  • CSS
  • Python
  • JavaScript
